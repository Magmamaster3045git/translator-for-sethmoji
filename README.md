# Sethmoji Translator

An offline Android app for translating English into the Sethmoji language.

## Features

- English → Sethmoji translation
- Sethmoji → English translation
- Emoji recognition
- Editable dictionary
- Search
- Offline storage
- CSV import/export
- Material Design interface

## Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- Android SDK
- GitHub Actions
